# coreset_softlabel
Code base for TMLR paper "Rethinking Coreset Selection: The Surprising Effectiveness of Soft Labels" to study of impact of soft labels on various coreset selection algorithms.

We utilize the below mentioned repositories publicly available by their corresponding authors.

1. D2 Pruning : https://github.com/adymaharana/d2pruning
2. DeepCore: https://github.com/PatrickZH/DeepCore
